# 🏠 MaidCare – Home Services Management System

MaidCare is a web-based platform that connects users with verified housekeepers for daily household services. It provides a seamless experience for users to request services and for admins to manage and assign housekeepers efficiently.


---

## 🚀 Features

### 👤 User

* Register & Login
* Request housekeeper services
* Auto assignment of housekeepers
* View personal requests
* Profile management

### 🛠️ Admin

* Admin login dashboard
* Manage categories
* Manage housekeepers
* View & assign requests
* Cancel requests

### ⚡ System Features

* Role-Based Access (Admin/User)
* Smart housekeeper assignment (based on availability & workload)
* Duplicate request prevention
* Session-based authentication
* REST APIs for external usage

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Database:** MongoDB
* **Frontend:** HTML, Bootstrap
* **API:** Django REST Framework
